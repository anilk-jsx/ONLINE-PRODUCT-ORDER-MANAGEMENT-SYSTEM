Put product photos in this folder.

Best format: JPG or PNG.
Recommended size: 600x600 or 800x800.

File name rule:
1. Convert product name to lowercase.
2. Replace spaces and symbols with hyphen.

Examples:
apple-iphone-16-pro-max.jpg
samsung-galaxy-s25-ultra.jpg
sony-wh-1000xm6.jpg
hm-premium-cotton-shirt.jpg

If exact product photo is missing, the website shows a lightweight fallback image so it will not hang.
