# QuickMart - Online Product Order Management System

Professional full-stack OMS built with Angular, Spring Boot and MySQL.

## Login Credentials

Admin: admin@quickmart.com / admin123  
Customer: customer@quickmart.com / customer123

## Included Modules

- Login and customer registration
- Admin dashboard with revenue in Indian rupees
- Customer shop and product booking
- Product management with category, GST, price and stock
- Stock add/remove/adjust workflow
- Order status management
- Delivery partner and tracking management
- GST invoice preview and print
- Customer issue tickets and admin replies
- Reports for sales, category stock and customer orders
- MySQL schema, Postman collection and screenshots folder

## Run

Backend:
```bash
cd backend
mvn spring-boot:run
```

Frontend:
```bash
cd frontend
npm start
```
